# Hướng dẫn hiển thị sơ đồ (Flowchart) trong VS Code

Các file trong thư mục này sử dụng cú pháp **Mermaid** để vẽ sơ đồ trực tiếp trong Markdown. Để xem được hình ảnh thay vì code, bạn cần cài đặt các Extension sau trên VS Code:

## 1. Extension cần thiết (Bắt buộc)
*   **Markdown Preview Mermaid Support** (của Matt Bierner): Giúp trình xem trước (Preview) của VS Code hiểu và vẽ được sơ đồ Mermaid.
    *   ID: `bierner.markdown-mermaid`

## 2. Extension hỗ trợ (Khuyên dùng)
*   **Mermaid Editor**: Hỗ trợ viết code Mermaid có gợi ý và xem trực tiếp.
    *   ID: `tomoyukim.vscode-mermaid-editor`
*   **Markdown All in One**: Cải thiện toàn diện trải nghiệm viết Markdown.
    *   ID: `yzhang.markdown-all-in-one`

---

## Cách xem sơ đồ:
1. Mở một file bất kỳ (ví dụ: `hinh1_kien_truc_luong_du_lieu.md`).
2. Nhấn tổ hợp phím `Ctrl + Shift + V` để mở cửa sổ **Preview**.
3. Sơ đồ sẽ được tự động vẽ và hiển thị dưới dạng hình ảnh.

## Cách xuất ra file ảnh (PNG/SVG):
Nếu bạn đã cài **Mermaid Editor**, bạn có thể chuột phải vào đoạn code Mermaid và chọn các tùy chọn Export để lưu sơ đồ thành file ảnh phục vụ cho báo cáo.
